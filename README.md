# mf-accounting-csv-analyzer

会計ソフト（マネーフォワード クラウド確定申告）からエクスポートした会計データを分析・検査します。

データはGUIからインポートし、`data/{年度}/` に配置して使います（Git管理対象外）。

独立したツールではなく、このプロジェクトフォルダー内で完結する運用を想定しています。

## 必要環境

- Node.js 22+
- 最新のWebブラウザ
- pnpm
- Python 3
- [uv](https://docs.astral.sh/uv/)（テスト実行に必要）

## セットアップ

```bash
pnpm install
```

## CSV インポート

```bash
pnpm start
```

ブラウザーで <http://localhost:3456> を開き、会計ソフトからエクスポートしたCSVをドラッグ＆ドロップします。

- Shift-JIS は自動で UTF-8 に変換
- ファイルタイプ（仕訳帳 / 総勘定元帳）はヘッダーから自動判定
- 年度は取引日から自動判定
- `data/{年度}/` に保存（`data/` は `.gitignore` 済み）

## 分析

`check/` フォルダー内のスクリプトなどを使いつつ、基本AI任せで。
