# schema/ — CSVスキーマ定義

会計ソフトからエクスポートされるCSVの種別定義を管理する。
PythonチェックスクリプトとGUIの両方がここを正本として参照する。

## ファイル構成

- `csv-types.json` — 対応するCSV種別の一覧。各エントリーが個別スキーマファイルを参照する。
- `journal.json` — 仕訳帳CSVのスキーマ（カラム名、保存ファイル名、日付カラム）。

## スキーマの形式

各スキーマJSONは以下のフィールドを持つ。

- `saveName` — インポート時の保存ファイル名（例: `"仕訳帳.csv"`）
- `dateColumn` — 日付が入るカラム名（年度判定に使用）
- `columns` — カラム名の配列（CSVヘッダーとの照合に使用）

## CSV種別の追加手順

1. `schema/<name>.json` を作成（上記の形式に従う）
2. `csv-types.json` にエントリーを追加
3. Python側で必要なら `analysis/` に定数・補助モジュールを追加し、チェック本体は `analysis/checks/` に配置
